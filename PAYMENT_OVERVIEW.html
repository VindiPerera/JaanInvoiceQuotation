How to Use

For End Users

  1. Open invoice detail page
  2. Click blue "Record Payment" button in header
  3. Fill in modal: date, amount, method, optional reference & notes
  4. Click "Record Payment" to save
  5. Payment appears in history table with user who recorded it
  6. Invoice totals update automatically
  7. Download payment receipt PDF when needed

For Developers

use App\Services\PaymentService; $paymentService = app(PaymentService::class); // Record payment with validation $payment = $paymentService->recordPayment($invoice, [ 'payment_date' => '2026-07-10', 'amount' => 50000, 'payment_method' => 'bank_transfer', 'reference_number' => 'TXN-123456', 'notes' => 'Customer payment' ]); // Get summary statistics $summary = $paymentService->generatePaymentSummary($invoice); // Check payment status if ($invoice->isFullyPaid()) { // Mark as paid in other systems }
